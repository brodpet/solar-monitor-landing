# Solar Monitor — Download Page

Static landing page for distributing the Solar Monitor Android APK.

- `index.html` — the download page; the button links to the APK hosted on Google Drive.
- To ship a new APK version: upload the new file to Google Drive (or use "Manage versions" on the existing file to keep the same link), then update the version text in `index.html`.

Deploy on Vercel: import this repo, no build settings needed (plain static site).
