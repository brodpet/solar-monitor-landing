# Solar Monitor — Download Page

Static landing page for distributing the Solar Monitor Android APK.

- `index.html` — the download page; the button links to the APK attached to this repo's GitHub Releases.
- To ship a new APK version: create a new release (e.g. `v1.0.7`) with the APK attached as `solar-monitor.apk`, then update the button URL and version text in `index.html`.

Deploy on Vercel: import this repo, no build settings needed (plain static site).
